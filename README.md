# Baby Theo Runner - Web Canvas

Versao HTML5 nativa, sem pygame e sem pygbag.

## Testar localmente

Dentro desta pasta, rode:

```bash
python -m http.server 8000
```

Abra:

```text
http://localhost:8000
```

## Celular na mesma rede

Descubra o IP do PC com `ipconfig` e abra no celular:

```text
http://IP_DO_PC:8000
```

## Publicar

Suba esta pasta em itch.io, Netlify, Vercel ou GitHub Pages.

## Controles

Desktop: setas ou A/D.  
Celular: toque no lado esquerdo ou direito da tela.

O ranking usa localStorage do navegador.
